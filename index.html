Passos para Hospedar um Projeto no Github Pages

  1. Crie um repositorio no Github com mesmo nome do seu Projeto
  2. No terminal,inicialize o repositorio local com git init
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos com git.add . faça o commit com git commit -m "first commit"
  5. Concet ao repositorio remoto com: git remote add origen https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. Envie os arquivos com git push -u origin main
  7. No Github, vá em settings > Pages
  8. Em "Branch", selecione main e a pasta / (root). Clique em save
  9. Apos alguns segundos, seu site estará disponivel no link exibido em Pages
